# Virtual Gamepad Emulator

**Проект не поддерживается, код предоставлен как есть. Может потребоваться адаптация под вашу систему.**

Эмуляция виртуального геймпада Xbox 360 с клавиатуры. Нажатия клавиш преобразуются в кнопки геймпада (D-pad и основные кнопки). Создано для игры в The Binding of Isaac в кооперативе, когда не было физического геймпада. Также я понимаю, что код далёк от идеала, так как понадобился для одной игровой сессии, и было желание написать это самому. 

## Как работает

- Использует библиотеку `vgamepad` для создания виртуального геймпада.
- Библиотека `keyboard` перехватывает глобальные нажатия клавиш.
- Нажатия клавиш `W, A, S, D` → стрелки геймпада (D-pad).
- Нажатия `↑, ↓, ←, →` → кнопки A, B, X, Y.
- Прочие клавиши (пробел, enter, tab, q, e, ctrl) → триггеры и бамперы.

- Windows (библиотека `vgamepad` работает только на Windows)
- Установленный Python 3.7+
